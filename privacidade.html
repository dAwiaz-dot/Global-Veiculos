Privacidade

Política de privacidade

Como a Global Veículos trata os dados enviados pelos canais de contato.

Dados coletados

O site pode solicitar nome, telefone, veículo de interesse e mensagem para facilitar o atendimento comercial pelo WhatsApp.

Uso das informações

As informações enviadas são usadas para responder dúvidas, simular financiamento, avaliar troca e dar continuidade ao atendimento solicitado pelo cliente.

WhatsApp e Instagram

Ao clicar em botões de WhatsApp ou Instagram, o cliente passa a usar plataformas de terceiros, sujeitas às políticas próprias dessas empresas.

Armazenamento

Esta versão do site não possui banco de dados online. O painel administrativo salva o estoque localmente no navegador usado pela loja.

Solicitações

Para solicitar correção ou remoção de dados enviados pelo atendimento, fale com a Global Veículos pelo WhatsApp oficial.
